My New Website!

15.02.26

I finally decided to revamp my website. Here's why I did it and what I'll be using this new one for.

My first portfolio website was inpsired by one of my favourite games, Alien: Isolation. Read more about it here.
It had limited functionality with only 3 pages but I'm still really proud of it. Now is the time to transition into something that I can use to document my work as a student and a developer. I'm hoping to work on some projects focused on cybersecurity and I'm excited to share my progress here.

This new site is built with pure HTML and CSS. No frameworks, no bloat, just code. It takes me back to the basics and allows me to have full control over every pixel.

I've also decided to use GitHub Pages for hosting. It's free, fast, and integrates perfectly with my workflow. Plus, it pushes me to commit more often!

I'm looking forward to starting this new chapter and I can't wait to share my journey with you all.
