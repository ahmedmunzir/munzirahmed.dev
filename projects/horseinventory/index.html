HorseInventory 1.21

26.09.24 GitHub

A plugin for 1.21 Minecraft written in Java. Adds a simple GUI for a horse inventory. Horse needs to be tamed and wearing a saddle for it to work.

The two week Minecraft phase is a yearly occurrence for all Minecraft fans. One day, as I was playing, I realised that it would be pretty useful to have your horse's saddle come equipped with some extra pockets just like in real life.
This, however, is not a feature in vanilla Minecraft, so I decided search online for any solutions. Unfortunately, there were none compatabile with the version of Minecraft I was currently playing (1.21), so I decided to make one myself.

How it works


The plugin works by listening for the right-click event on a tamed horse with a saddle. When this event is triggered, the plugin opens a GUI for the player to store items in the horse's inventory.

  1. Tame a Horse

  2. Put a saddle on the horse

  3. Shift + right-click the horse to open the inventory GUI
